https://bored-api.appbrewery.com/ENDPOINT?QUERY=VALUE&QUERY=VALUE

https://bored-api.appbrewery.com/filter?type=education

https://bored-api.appbrewery.com/ENDPOINT/{PATH-PARAMETERS}
